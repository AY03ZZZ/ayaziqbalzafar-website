# Ayaz Zafar Portfolio

Modern static personal portfolio for Ayaz Zafar, built with plain HTML, CSS, and JavaScript. The design uses a compact resume-style layout with a black, white, and `#007fff` colour palette.

## Files

- `index.html` - semantic page structure, SEO metadata, Open Graph tags, and all sections.
- `styles.css` - responsive compact portfolio design, dark/light themes, animations, and layout.
- `script.js` - theme toggle, scroll reveal, and active dock navigation.
- `assets/favicon.svg` - site favicon.
- `assets/og-image.svg` - social sharing preview image.
- `assets/profile-placeholder.svg` - temporary circular profile image.
- `assets/icon-*.svg` - small static row images for experience, qualifications, and education.

## Profile Source

The visible portfolio content is based on publicly indexed LinkedIn information for Ayaz Iqbal Zafar, including Appstack, Dudley College of Technology, City & Guilds ICT Systems and Principles qualifications, Elmhurst Energy certification, and Tipton location.

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
npx serve .
```

## Deploy

This site is ready for static hosting.

### Vercel

1. Create a new Vercel project.
2. Import this folder/repository.
3. Use the default static settings.
4. Deploy.

### Netlify / GitHub Pages

Upload or publish the folder as a static site. No build command is required.
